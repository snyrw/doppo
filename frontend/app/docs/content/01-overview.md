title: Overview
---

Doppo is a sandbox to run certain mechanistic interpretability techniques on a wide range of TransformerLens models. We try to streamline workflows and provide QoL improvements over manually writing/running code. This is done by maintaining notebook standards while making the best of the flexibility and speed our format provides.

If you are new to interpretability, the [tutorial](/tutorial) walks through the IOI circuit on GPT-2 Small and a separate steering example. It can be completed in its entirety without an account as all of the results are pre-computed.

Past that, you can create an account with us if you'd like to start using live compute. GPU time is billed from a credit balance, and every account gets a $1/month grant priced against our inference provider, Modal (see [Credits and pricing](#credits-and-pricing)). Repeating a run with identical settings hits a cache and is free.
