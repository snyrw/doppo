title: Overview
---

Doppo is a sandbox to run certain mechanistic interpretability techniques on a wide range of TransformerLens models. We try to streamline workflows and provide QoL improvements over manually writing/running code. This is done by maintaining notebook standards while making the best of the flexibility and speed our format provides.

These docs cover some of the basics from techniques, model coverage, pricing, how and what we cache, intentional limits we've set, and how to contact us. A lot of this is simple plaintext, and rather than caveating it redundantly, it should be mentioned that many parts of this project are subject to change.
